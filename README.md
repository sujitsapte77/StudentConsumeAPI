# Window API DataGridView
This is window form application that consumes an ASP.NET Core Get API and displays the data in a DataGridView.

## Technology Used
-ASP.Net Core API
-Windows Forms
-HttpClient
-NewtonSoft.Json

## Features
-Consume ASP.NET Core Get API
-Display JSON response in DataGridView
-Asynchronous API calls using HttpClient
